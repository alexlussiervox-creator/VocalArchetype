# VoxTools

Clean GitHub-ready React + Vite source for a mobile-first vocal prompt builder.

## Stack
- React 18
- Vite 5
- Node 20

## Commands
```bash
npm install
npm run dev
npm run build
```

## Expected UX defaults
- Prompt Builder open by default
- Descriptor Library closed by default
- Artist Profiles closed by default
- Vocal Archetypes closed by default
- Descriptor subcategories closed by default
- Reset restores default builder state
- Cinematic Surge present
- Artists and Archetypes sections collapsible


Updated build: reset now returns the builder to an empty state and the prompt area starts blank.
